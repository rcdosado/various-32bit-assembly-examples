this package includes
---------------------

link.exe microsoft's linker 32
ml.exe   macro assembler 32
tasm32.exe borland's turbo assembler 32
tlink32.exe borland's turbo linker 32
imsvcrt.lib standard lib functions,e.g printf,scanf,etc..
import32.lib windows functions,e.g ExitProcess

they are needed for compiling the assembly sourcecodes
